 PG / Property Management System

A full-stack web application developed to manage PG/property operations including tenant records, rent tracking, complaints, and property management.

 Technologies Used
- React / Next.js
- Supabase
- SQL Database
- Tailwind CSS

 Features
- Secure login & authentication
- Tenant & property management
- CRUD operations
- Complaint handling
- Dashboard & record tracking

 Project Type
Client-based real-world project developed based on PG owner requirements.
